# How to update Node.js on centos 6.X

<br/>

    # node -v
    v0.10.40

    # npm cache clean -f
    # npm install -g n

    # /usr/lib/node_modules/n/bin/n latest

    # ln -sf /usr/local/n/versions/node/<VERSION>/bin/node /usr/bin/node

    # node -v
    v6.2.2
